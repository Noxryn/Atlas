# 数组
数组 - 相同数据元素的集合
注：数据元素类型一致、内存位置连续
一维数组语法：
1. 数据类型 数组名[len]; 
2. 数据类型 数组名[len] = {,};
3. 数据类型 数组名[] = {,};
二维数组语法：
1. 数据类型 数组名[row][col];
2. 数据类型 数组名[row][col] = {{,}, {,}};
3. 数据类型 数组名[row][col] = {, , , , };
4. 数据类型 数组名[][col] = {, , , , };
   
# 函数
函数 - 封装常用代码，减少重复代码
语法 - 返回值类型 函数名 (参数列表) {  //*函数体*// return 表达式 }  
调用 - 函数名称(实参) --> 实参传值到形参

# 指针
指针 = 地址
语法：`数据类型 *指针变量名 = &变量名`
解引用： `* 指针变量名` 
指针占4字节（32）, 8字节（64）
常量指针 - const p - 值不能改变
指针常量 - p const - 指向不能改变
