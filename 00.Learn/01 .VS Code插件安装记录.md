# 美化类
## 0.GitHub Theme
VS Code美化主题。可以让VS Code更好看。
这个是最经典的主题之一，还有一个Dracula Theme我也很喜欢
## 1.vscode-icons
VS Code美化插件，更改不同扩展名文件的显示图标
# 基础类
## 0.C/C++ Extension Pack
VS code 中C/C++语言的环境配置插件。
在VS Code中编写C/C++语言的代码所用，这是一个整合包，包含：
1. 拓展支持C/C++语言的C/C++(***写C/C++必安***，需要进行配置)
2. 美化C/C++代码主题的C/C++ Themes (这个我卸了，省点空间)
3. 支持CMake的CMake和CMake Tools (多文件编程使用)  
可以单独安装你需要的插件，直接整合包可以少点几下安装 
## 1.Python
VS Code提供Python语言基础

# 工具类
## 0.Chinese (Simplified)
这个不多说，汉化插件。
如果你自信英语水平了得或者是想适应纯英文开发环境可以不安。
*(非必需)*
## 1.Live Server
打开浏览器，实时预览html开发的效果。
(还有一个open in browser，但这个插件修改代码后要手动刷新)
## 2.Competitive Programming Helper (cph)
OJ刷题使用，配合Competitive Companion 浏览器插件可以在实现样例的自动输入验证，提高刷题效率
## 3.Bookmarks
书签跳转，提升阅读代码的效率
## 4.Doxygen Documentation Generator
提供Doxygen格式的注释 
## 5.Markdown
由于不想为写博客单独在下一个typlora,配置vscode支持markdown语法，具体插件如下：  
Markown All in one ：提供Markdown语法支持  
Markdown Preview Github Styling ：实时预览  
Markdown Emoji ：emoji表情包支持 (这个就看各人了)
## 6.Remote-SSH
本地vscode连接远程主机使用，需要进行配置。
## PicGo
图床管理工具，有独立的软件版。用于保存markdown文件显示的图片,需配置图床
