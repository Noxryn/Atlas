# C/C++
C语言或许是科班程序员学习的第一门语言，也是最没学好的一门语言。  
本人学习的第一门编程语言就是C,当时没有自学的意识，单纯跟随老师的讲解。大一时做个管理系统甚至连do while循环都不会使用:smile:   
注：
- 由于C++是C的超集，基础内容一致，一般合并介绍
- 入门其实可以直接学习C++，但有时间还是建议先学C，循序渐进更易理解
- C是面向过程程序语言，C++是面向对象程序语言，它们的具体区别就自行了解了
  
## 1.程序设计入门——C语言

## Git
Git只要会提交到远程库，实践中多使用也就熟练了，不必过多关注
  
### 1.尚硅谷Git入门到精通全套教程 2021版
记录：
- 讲述了常用的Git命令和使用方法，足够正常使用
- GitHub、Gitee、GitLab三个平台都有涉及
- 截至撰写时还有个23版，但基于客户端软件，个人认为没有必要
- 视频快速过一遍或者跟着做，git学习还是动手实操更重要
- 这个视频是本人学习时观看，其它同理，主要关注命令的使用及作用效果
[资源链接](https://www.bilibili.com/video/BV1vy4y1s7k6/)  
  
### 2.Learn Git Branching <网站>
记录：
- 一个学习Git的在线网站，游戏闯关类型，同名称主要涉及分支相关内容
- 可以动态观看到每一条命令的效果，快速了解指令作用
- 这个看个人需求吧，有时间可以玩玩
[资源链接](https://learngitbranching.js.org/?locale=zh_CN)
  
### 3.《GitHub漫游指南》
记录：
- 作者以自身经历介绍了GitHub的作用，使用方法，代码测试等相关内容
- 如果你是一个此前从没接触过GitHub的coder可以浏览一下
[资源链接](https://github.phodal.com/)
  
## Markdown
Markdown 是一种轻量级的标记语言，可用于在纯文本文档中添加格式化元素。广泛应用在开源项目，博客文章等。
   
### 1.何方-Markdown胎教级教程
记录：
- 本人学习Markdown时参考的视频
- Markdown语法并不复杂，多用即可熟练
- 个人认为学习Markdown不用观看视频
[资源链接](https://www.bilibili.com/video/BV1hG4y1H7iZ/)
  
### 2.Markdown官方教程<网站>
记录：
- 中文官网，全面介绍了Markdown语法
- 更推荐只看语法后自己去练习
[资料链接](https://markdown.com.cn/)
