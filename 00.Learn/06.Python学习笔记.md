# 1.数据基本类型
## 整型 int
- 不限制大小  
- 运算  

|运算符|功能|备注|    
|-|-|-|  
|m+n|加法|| 
|m-n|减法||  
|m*n|乘法||  
|m//n|整除|结果为商的整数部分|  
|m/n|除法|结果为小数|  
|m%n|取余||  
|divmod(m,n)|整除+取余|m//n,m%n|
|m**n|乘方|m的n次方|  
|abs(m)|绝对值|| 
|大小比较支持连续比较|a > b > c|true|

## 浮点数 float
- 17位有效数字(C++ 7,16)

## 复数
- 内置复数类型
- real + imag*j (1+2j)
- 复数只能比较是否相等

## 逻辑类型 bool
True OR False
not > and > or
空值 - None

## 文本
- 使用单引号或双引号包围字符串
- 三引号包围带有换行的字符串
- 字符串下标编号正向（0~n）,反向（-1~-n）
